# التوجيه والإرشاد — بوابة الإرشاد الطلابي

موقع عربي متجاوب للمبادرات الإرشادية والمسابقات والفعاليات والإحصائيات.

## التشغيل

المشروع موقع ثابت، ويمكن فتح `index.html` مباشرة أو تشغيله بخادم محلي.

## النشر على Cloudflare Pages

- ارفع الملفات إلى مستودع GitHub.
- اربط المستودع من Cloudflare Pages.
- لا يوجد أمر بناء.
- اجعل مجلد النشر هو مجلد المشروع الرئيسي.

النتائج في النسخة الحالية محفوظة محليًا في متصفح المستخدم. توحيد النتائج بين جميع الطلاب يتطلب ربط Cloudflare D1 عبر Worker أو Pages Functions.
